jvm-command
===========

Command is `system`, then use tab to list the available sub-commands.


Ex:

Setting a property

     > system setProperty logback.debug true
     [info] Setting logback.debug = true


Memory stats

     > system memoryStats
     [info] Memory
     [info]   total = 275.63 MB
     [info] 	 free = 134.19 MB
     [info] 	 used = 141.50 MB
     [info] 	 max = 1.33 GB
